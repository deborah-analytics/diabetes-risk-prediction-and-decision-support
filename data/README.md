# Data Description

This project uses the BRFSS 2021 Health Indicators dataset to support both multiclass diabetes risk prediction and a clinical decision-support application.

## Dataset Overview

The dataset contains demographic, lifestyle, and clinical health indicators used to predict diabetes status (No, Prediabetes, Diabetes).

## Source

* Behavioural Risk Factor Surveillance System (BRFSS), 2021

## Features

* BMI
* Age
* High Blood Pressure
* High Cholesterol
* Physical Activity
* Smoking Status
* General Health
* Income
* Education
* and other health-related indicators

## Target Variable

* Diabetes_012:

  * 0 = No Diabetes
  * 1 = Prediabetes
  * 2 = Diabetes

## Note

The dataset is not included in this repository due to size and licensing considerations.
