# Documentation

This folder contains supporting documentation for the diabetes risk prediction and decision support project.

## Files

### project_report_summary.pdf
A concise summary of the project objectives, methods, results, and dashboard applications.

### methodology_overview.pdf
Describes the CRISP-DM process, preprocessing, resampling strategy, and model evaluation workflow.

### data_dictionary.pdf
Defines the variables included in the dataset and their meanings.

### references.md
Contains the key academic references, NICE guidance, and data sources used in the project.
